# Interactive Particle Morphing with Three.js

An interactive 3D particle morphing visualization built with Three.js and GLSL shaders.

## Features

- Real-time particle morphing between multiple geometric patterns
- Smooth transitions with easing
- Dynamic color palettes
- Starfield background with twinkling stars
- Nebula layers for atmospheric depth
- Post-processing bloom effect
- Responsive design
- Mouse/touch interaction

## How to Run

Simply open `index.html` in a modern web browser that supports WebGL.

## Technologies Used

- Three.js - 3D graphics library
- GLSL Shaders - Custom vertex and fragment shaders
- Post-processing effects (Bloom)
- HTML5 Canvas
- ES6 Modules

## Patterns

The visualization cycles through four distinct patterns:

1. **Torus Knot** - A complex twisted torus shape
2. **Elliptical Orbits** - Multiple elliptical orbits with varying tilts
3. **Wave Orbit** - A sinusoidal wave orbit pattern
4. **Cosmic Web** - Organic, web-like structures

## Interaction

- Click or tap anywhere on the screen to morph to the next pattern.
- The camera orbits around the scene automatically.

## Browser Support

Requires a WebGL-capable browser. Tested on Chrome, Firefox, Safari, and Edge.
